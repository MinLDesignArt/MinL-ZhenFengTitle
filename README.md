二、设计定位
MinL振风标题体是一款以Source Han Sans的字形骨架为基础的中文标题字体，适合标题、包装、海报、展板、封面和品牌视觉中的大字号排版。主要是解决设计工作中，需要使用规范但又不刻板，不失设计感的黑体字。在设计工作中，有时所使用到的字体过于艺术化，不适合项目整体的风格，而默认的黑体字作为大标题或主要视觉中又可能太过规整，故开发了这款MinL振风标题体,在Source Han Sans的基础上做出克制的修改，保留核心的特色，并让文字整体更加醒目、柔和，略微增加艺术感，克制不张扬。解决每次使用默认的一些字体时就需要手动修改调整的重复工作。


主要视觉特征：
1. 字形直立、厚重，整体结构规整。
2. 采用宽版但较克制的小字面比例。
3. 横笔具有柔和的贝塞尔曲线张力。
4. 竖笔保持直立，并带有轻微、对称的内收感。
5. 横竖交接与部分转角采用柔和的小圆角过渡。
6. 拉丁字母和数字经过均衡字距调整。
7. 支持横排和竖排，包含竖排标点替代规则。
